# 🐍 Lógica de Programación con Python

Repositorio de ejercicios y ejemplos prácticos desarrollados durante un curso de lógica de programación usando Python como lenguaje base. Cada archivo cubre un tema específico, con explicaciones, ejemplos de uso y un ejercicio extra de aplicación.

---

## 📁 Estructura del repositorio

| Archivo | Tema |
|---|---|
| `01_syntax_date_types.py` | Sintaxis, variables y tipos de datos |
| `02_operators_control_structures.py` | Operadores y estructuras de control |
| `03_functions.py` | Funciones |
| `04_data_structures.py` | Estructuras de datos |
| `05_strings.py` | Cadenas de texto |
| `06_value_reference.py` | Valor y referencia |
| `07_recursion.py` | Recursividad |
| `08_stacks_queues.py` | Pilas y colas |
| `09_classes.py` | Clases |
| `10_inheritance_polymorphism.py` | Herencia y polimorfismo |
| `11_exceptions.py` | Manejo de excepciones |
| `12_file_handling.py` | Manejo de ficheros |
| `13_json_xml.py` | JSON y XML |
| `14_unit_tests.py` | Pruebas unitarias |
